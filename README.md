# AIML-Task-3-Linear-Regression
Implement and understand simple &amp; multiple linear regression. 

The objective of this project is to understand and implement both Simple Linear Regression and Multiple Linear Regression using Python and Scikit-learn. The project uses a housing dataset to predict house prices based on various property features and evaluates the model's performance using standard regression metrics.
The dataset contains housing information such as area, number of bedrooms, bathrooms, stories, parking availability, furnishing status, air conditioning, and other property-related features. The target variable is the house price.

Uploaded and loaded the housing dataset.
Explored dataset structure and features.
Checked for missing values.
Converted categorical features into numerical form using one-hot encoding.
Implemented Simple Linear Regression using area as the predictor.
Visualized the regression line.
Implemented Multiple Linear Regression using all available features.
Evaluated model performance using MAE, MSE, and R² Score.
Analyzed feature coefficients to understand feature importance.
Compared actual and predicted house prices using visualization.

Results and Findings
House area showed a positive relationship with price in Simple Linear Regression.
Multiple Linear Regression provided better predictive performance by utilizing all available features.
Features such as bathrooms, parking availability, air conditioning, and preferred area contributed positively to house prices.
The model successfully learned housing price trends and generated reliable predictions.
R² Score demonstrated the effectiveness of the regression model in explaining variations in housing prices.

This project demonstrates the practical implementation of regression techniques for house price prediction. By comparing Simple and Multiple Linear Regression models, it highlights the importance of using multiple relevant features to improve prediction accuracy and better understand factors influencing housing prices.
